# EchoForgeX Website

Marketing and blog website for EchoForgeX LLC - an AI-focused software studio and product company.

## Tech Stack

- **CMS:** WordPress 6.x
- **Theme:** GeneratePress Premium + custom child theme
- **Page Builder:** GenerateBlocks Pro
- **Hosting:** Linode (shared server with SignShield)
- **CDN/DNS:** Cloudflare
- **Email:** Hostinger

## Project Structure

```
echoforgex-website/
├── wp-content/
│   ├── themes/
│   │   └── echoforgex-theme/    # Custom child theme
│   └── plugins/                  # Custom plugins (if any)
├── docs/                         # Project documentation
├── assets/
│   ├── logos/                    # Logo files (SVG, PNG)
│   ├── images/                   # Source images
│   └── fonts/                    # Custom fonts (if any)
└── README.md
```
